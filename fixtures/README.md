# Données officielles (préfectures)

Les fichiers officiels du **Code Officiel Géographique (COG)** sont publiés par l'INSEE chaque année.

Sources recommandées :
- Jeu de données COG (data.gouv.fr) : https://www.data.gouv.fr/datasets/code-officiel-geographique-cog
- Page INSEE du millésime COG 2025 (CSV) : https://www.insee.fr/fr/information/8377162
